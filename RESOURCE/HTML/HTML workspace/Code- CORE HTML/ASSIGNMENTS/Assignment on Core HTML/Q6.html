HTML Tags

  1. <html> - Defines the root element of an HTML document.

  2. <head> - Contains information about the document, such as the title, author, and keywords.

  3. <title> - Specifies the title of the document.

  4. <body> - Contains the main content of the document.

  5. <p> - Defines a paragraph.

  6. <br> - Defines a line break.
