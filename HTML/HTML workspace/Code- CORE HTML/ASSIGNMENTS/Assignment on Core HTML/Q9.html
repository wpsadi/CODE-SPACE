1. <html>
    • USE CASE : Defines the root element of an HTML document.
    • BEST PRACTICE : The <html> tag should always be the first tag in an HTML document.

  2. <head>
    • USE CASE : Contains information about the document, such as the title, author, and keywords.
    • BEST PRACTICE : The <head> tag should always be closed before the <body> tag.

  3. <title>
    • USE CASE : Specifies the title of the document.
    • BEST PRACTICE : The title should be a short, descriptive string that accurately reflects the content of the document.

  4. <body>
    • USE CASE : Contains the main content of the document.
    • BEST PRACTICE : The <body> tag should always be closed after the <html> tag.
